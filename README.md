# 📱 QR Playground - Generator & Scanner

A lightweight, fully client-side web application to generate QR codes from text/URLs and scan existing QR codes using your device camera. Built with pure HTML, CSS, and JavaScript — **zero backend, zero dependencies, and it never sleeps** (deployed as a static site on Vercel).

🔗 **Live Demo:** [https://cibera-barcode.vercel.app](https://cibera-barcode.vercel.app)

## ✨ Features

- **🚀 Generate QR Codes:** Instantly generate QR codes from any text or URL.
- **📥 Download PNG:** Save generated QR codes as high-quality `.png` images with one click.
- **📷 Scan QR Codes:** Use your device camera (front/back) to scan any QR code.
- **🔄 Auto-Stop Scanner:** The scanner automatically stops after a successful scan to save battery.
- **📜 Local History:** Automatically saves your generation history in your browser's Local Storage (no database required).
- **📱 Fully Responsive:** Works perfectly on desktop, tablet, and mobile devices.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **QR Generation:** [QRCode.js](https://github.com/davidshimjs/qrcodejs) (CDN)
- **QR Scanning:** [html5-qrcode](https://github.com/mebjas/html5-qrcode) (CDN)
- **Hosting:** [Vercel](https://vercel.com) (Static Site)
